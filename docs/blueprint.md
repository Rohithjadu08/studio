# **App Name**: TruthSeeker

## Core Features:

- News Input: Allow users to input news articles via URL or text.
- Fake News Detection: Utilize a generative AI model to analyze news content and identify potential fake news indicators.
- Credibility Score: Provide a credibility score or rating based on the AI analysis.
- Source Analysis Tool: Employ an AI tool that analyzes the source of the news article to provide insights into its reliability and potential bias.
- Fact-Checking Report: Generate a detailed fact-checking report highlighting any false or misleading information.
- Corrective News: If fake news is detected, provide links to articles that present accurate and verified information.
- User Feedback: Allow users to provide feedback on the accuracy of the analysis and suggest improvements.

## Style Guidelines:

- Primary color: Navy blue (#2E5EAA) for trustworthiness and stability.
- Background color: Very light blue (#EBF4FA), providing a clean and unobtrusive background.
- Accent color: Green (#4CAF50) to indicate verified or corrected information; a contrasting shade of red could flag potentially false info.
- Body and headline font: 'Inter' for a modern, objective feel.
- Use simple, clear icons to represent different categories of analysis and types of information.
- Clean and organized layout with clear visual hierarchy to emphasize the most important information.
- Subtle animations for loading states and transitions to enhance the user experience.
